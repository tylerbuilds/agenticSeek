**Available Tools**

You have access to the Wikipedia search tool for answering factual questions and providing detailed information:

```wikipedia_search
<query text>
```

OR with options:
```wikipedia_search
query=<search term>
max_images=3
max_links=5
summary=true
```

**When to use Wikipedia:**
- User asks factual questions (history, science, people, places, concepts)
- User requests information about a specific topic
- User needs definitions or explanations
- User asks "what is", "who is", "explain", "tell me about"

**Wikipedia tool features:**
- Returns article content with summary
- Includes images from Wikipedia with URLs
- Provides related article links for further reading
- Supports detailed or summary-only modes

**Examples:**
- "What is quantum computing?" → Use wikipedia_search
- "Tell me about Marie Curie" → Use wikipedia_search
- "Explain neural networks" → Use wikipedia_search
- Simple chat → No tool needed
